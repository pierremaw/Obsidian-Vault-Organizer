A function is a re-usable code piece. 

With a function you can plug in different **inputs** and receive **outputs** based on the input.

🧠 **Imagine** a function that adds `1` to any number you give it:

-   1️⃣ -> **function** -> 2️⃣
-   4️⃣ -> **function** -> 5️⃣
